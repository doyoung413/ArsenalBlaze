# Arsenal Blaze : 커스텀 2D 엔진 기반 슈팅 액션 게임 데모

> 프로젝트 기간: 미상 (Tech Demo)
> 프로젝트 성격: 자체 엔진 개발 및 게임 로직 구현 데모

이 프로젝트는 C++와 SDL2, FMOD를 기반으로 자체 제작된 2D 게임 엔진을 활용한 탑다운/종스크롤 슈팅 게임 프레임워크입니다. 주로 다양한 적 탄막 / 무기 교체 시스템과 인게임 상점을 통한 기체 업그레이드를 지원하며, 싱글톤 매니저와 컴포넌트 기반 아키텍처로 짜여진 견고한 커스텀 엔진 구조를 갖추고 있습니다.

## ✨ 주요 기능 (Key Features)

### 1. 커스텀 2D 게임 엔진 (Core Engine)
* **컴포넌트 기반 구조 (ECS 형태)**: PlayerComponent, Physics 등 게임 오브젝트의 동작과 물리 처리, 상태를 독립적이고 모듈화된 컴포넌트로 관리합니다.
* **싱글톤 통합 매니저 (Instance Hub)**: Sprite, Object, Input, Sound, Particle 등 주요 시스템 매니저들을 중앙 허브 인스턴스에서 통합 관리하여 씬(Level)과 객체들의 처리를 효율적으로 통제합니다.

### 2. 인게임 콘텐츠 및 시스템 (In-Game Systems)
* **다양한 무기 및 전투 시스템**: 일반 공격(Normal) 외에 레이저(Laser), 유도 미사일(Homing Missile) 등의 무기를 실시간으로 교체하며, 적들의 레이저, 폭발 미사일, 스프레드 탄막 패턴 등에 대응합니다.
* **상점 및 기체 업그레이드 체계**: 스테이지 간 진입 가능한 상점에서 재화(Money)를 소모하여 주력 무기의 화력을 높이거나 최대 체력을 증가시킬 수 있습니다. 방어용 베리어 등의 특수 장비 또한 장착을 계획 중입니다.
* **내장 맵 에디터**: 디버그 모드(`_DEBUG`)에서 구동 가능한 인게임 맵 에디터를 포함하고 있어, 별도의 툴 없이 레벨 데이터를 수정하고 즉각적인 테스트가 가능합니다.

## 🛠 기술 스택 (Tech Stack)

* **Language**: C++
* **Framework/API**: Custom 2D Engine
* **Libraries**: 
  * SDL2 (윈도우 창 생성, 2D 그래픽 렌더링, 사용자 입력 이벤트 처리)
  * FMOD (효과음, BGM 등 강력한 오디오 시스템 지원)
  * glm (벡터, 행렬 등 수학/물리 계산 지원 추정)
* **Tools**: Visual Studio

## 🚀 설치 및 실행 방법 (Installation & Setup)

1. **솔루션 열기**: Visual Studio에서 프로젝트 최상단에 있는 `ArsenalBlaze.sln` 솔루션 파일을 엽니다.
2. **시작 프로젝트 지정**: 솔루션 탐색기에서 `ArsenalBlaze` 프로젝트를 **시작 프로젝트**로 지정합니다.
3. **플랫폼 설정 (중요)**: 빌드 아키텍처 플랫폼을 **x64 (64비트)** 로 변경합니다. (32비트 환경에서는 컴파일이 지원되지 않습니다.)
4. **빌드 및 실행**: 디버그(Debug) 모드나 릴리즈(Release) 모드 상태에서 `F5` 또는 `Ctrl+F5`를 눌러 컴파일하고 게임을 실행합니다.

---

### 🕹️ 조작키 (Controls)
- **이동**: 방향키 (`↑`, `↓`, `←`, `→`)
- **공격 및 메뉴 선택**: `Z` 키 (메인/상점 화면에서는 `Enter` 혼용)
- **무기 교체 (Subweapon)**: `X` 키

---

### 🗺️ 내장 맵 에디터 (Map Editor)
본 프로젝트는 인게임에서 실시간으로 맵을 편집할 수 있는 에디터가 내장되어 있습니다. 디버그(`_DEBUG`) 빌드로 실행 중일 때만 사용할 수 있습니다.

#### 진입 방법
- 게임 플레이 중 **`M`** 키를 눌러 에디터 모드(`EDITOR`)와 플레이 모드(`RUNNING`)를 전환할 수 있습니다.

#### 에디터 조작키
- **카메라 이동**: `W`, `A`, `S`, `D`
- **오브젝트 선택 및 조작 (Modifier)**: 마우스 왼쪽 버튼 클릭
- **오브젝트 생성 (Creator)**: 마우스 가운데 버튼 (휠) 클릭
- **오브젝트 삭제**: 마우스 오른쪽 버튼 클릭
- **백그라운드 생성 (Background Creator)**: 마우스 가운데 버튼 또는 `Space` 키
- **모드 전환 창 (UI)**: 화면에 표시되는 ImGui 윈도우를 통해 `ObjectModifier`, `ObjectCreator`, `BackgroundCreator` 모드를 선택 혹은 세부 속성 조절을 할 수 있습니다.
